📧 Email Notifications — Setup Guide

⏱ Takes about 5 minutes. Free for up to 200 emails/month.
1
Sign up at EmailJS Go to emailjs.com → click "Sign Up Free" → create an account using your Gmail
2
Add an Email Service In EmailJS dashboard → click "Email Services" → "Add New Service" → choose Gmail → connect your globalhealthbridge.in@gmail.com account → click "Create Service". Copy the Service ID shown.
3
Create an Email Template Click "Email Templates" → "Create New Template". Set Subject to:
New Lead: {{lead_name}} — {{lead_specialty}}
In the body, paste these variables (EmailJS will fill them automatically):
Name: {{lead_name}}
Email: {{lead_email}}
Phone: {{lead_phone}}
Country: {{lead_country}}
Specialty: {{lead_specialty}}
Description: {{lead_description}}
Source: {{lead_source}}
Date: {{lead_date}}
Click "Save". Copy the Template ID.
4
Get your Public Key In EmailJS → click your account name (top right) → "Account" → copy the Public Key shown on that page.
5
Paste all 3 keys below and save Fill in the 3 fields below with your Service ID, Template ID and Public Key, then click Save & Test.
Enable email notifications

🔧 How the Lead Capture System Works

The Global Health Bridge website form sends leads to this dashboard automatically. Here's how everything connects:

1
Website Form SubmissionPatient fills out the contact form on globalhealthbridge.co.in and clicks Submit.
2
Lead Stored LocallyThe lead is saved to this dashboard using browser localStorage. The lead appears in the table instantly.
3
WhatsApp Alert SentYou receive an instant WhatsApp message with the lead's name, specialty, country, and email. Requires CallMeBot API key (free).
4
Email Notification SentA full email with all lead details is sent to globalhealthbridge.in@gmail.com. Requires EmailJS setup (free for 200/month).
5
Status ManagementYou log into this dashboard to review leads, read the full enquiry, and update the status to New → In Progress → Closed.

⚙️ Setup Checklist

✅ Download both HTML files (website + dashboard)
☐ Sign up at emailjs.com (free) → get Service ID, Template ID, Public Key
☐ Activate CallMeBot for WhatsApp (see Settings tab)
☐ Enter all keys in Settings tab → Save
☐ Upload both files to your GoDaddy hosting
☐ Test the form on the live website

Note: This dashboard stores leads in your browser's localStorage. For a production multi-user database, a future upgrade to a cloud backend (e.g. Firebase or Supabase) is recommended.
